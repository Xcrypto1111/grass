# getgrass
## installation
install python3

for windows: https://www.python.org/ftp/python/3.12.2/python-3.12.2-amd64.exe 

for linux distros: ```sudo apt-get install python3``` or ```sudo yum install python3``` or ```sudo dnf install python3```
